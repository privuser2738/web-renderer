How It Works

1. Include the SDK

<script src="web-rerender.js"></script>

2. Initialize

WebRerender.init({ pagination: { mode: 'intelligent' }, images: { quality: 'adaptive' } });

3. That's It!

The SDK automatically detects your device, analyzes your page content, and applies optimal rendering strategies. No additional configuration required.

Image Optimization

Images are automatically optimized based on your device and connection. The SDK applies lazy loading, quality adjustment, and provides user controls.

API Reference

Navigation Methods

WebRerender.goToPage(3); // Go to page 3 WebRerender.nextPage(); // Next page WebRerender.prevPage(); // Previous page WebRerender.getCurrentPage(); // Get current page number WebRerender.getTotalPages(); // Get total pages

Device Information

const device = WebRerender.getDeviceInfo(); const input = WebRerender.getInputInfo();

Image Control

WebRerender.setImageOptimization(false); // Disable WebRerender.setImageOptimization(true); // Enable

Events

Listen to SDK events to react to page changes, device changes, and more.

WebRerender.on('pagination:pagechanged', (e) => { console.log('Page changed:', e.detail); }); WebRerender.on('device:orientationchange', (e) => { console.log('Orientation changed:', e.detail); }); WebRerender.on('input:changed', (e) => { console.log('Input type changed:', e.detail); });
