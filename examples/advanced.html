API Examples

Navigation API

// Navigate programmatically WebRerender.goToPage(3); WebRerender.nextPage(); WebRerender.prevPage(); // Get page info const current = WebRerender.getCurrentPage(); const total = WebRerender.getTotalPages();

Device Detection

// Get device information const device = WebRerender.getDeviceInfo(); console.log(device.type); // 'desktop' console.log(device.os); // 'windows' console.log(device.browser); // 'chrome'

Input Detection

// Get input information const input = WebRerender.getInputInfo(); console.log(input.primary); // 'mouse' console.log(input.types.touch); // true/false

Event Listening

// Listen to page changes WebRerender.on('pagination:pagechanged', (e) => { console.log('Page:', e.detail.to); }); // Listen to input changes WebRerender.on('input:changed', (e) => { console.log('Input:', e.detail.to); });

Configuration Examples

Intelligent Mode (Current)

Breaks content at semantic HTML boundaries like sections, articles, and headings. This provides the most natural reading experience.

WebRerender.init({ pagination: { mode: 'intelligent', breakpoints: 'semantic' } });

Auto Mode

Automatically breaks content based purely on viewport height. Faster but less intelligent.

WebRerender.init({ pagination: { mode: 'auto', viewportMultiplier: 1.0 } });

Manual Mode

Use data attributes to define exact page boundaries. Perfect for precise control.

// HTML <div data-webrerender-page>Page 1</div> <div data-webrerender-page>Page 2</div> // JS WebRerender.init({ pagination: { mode: 'manual' } });

Emergency Mode

Emergency mode automatically detects and fixes problematic pages. It activates when:

  • Page height exceeds 10x viewport
  • More than 50 images detected
  • More than 10 videos/iframes
  • Z-index conflicts detected
WebRerender.init({ emergency: { detectConflicts: true, autoFix: true, videoOptimization: true } });
